Variables

If a certain string appears often in your regular expressions, you can assign to a variable. A variable declaration has a very simple syntax: 

<var name="var-name" value="string" />

You can then refer to a variable using a shell-style syntax: 

${var-name}

For example, you could define a variable for your application's base package, then use it in your patterns: 

<var name="app-base" value="com.vaporcorp.whatsitproject" />
<pattern name="ui" regex="${app-base}.ui.**" />
<pattern name="db" regex="${app-base}.db.**" />

As with patterns, the scope of a variable is the ruleset or foreach element which contains it.
